#用途：
    可以录制电脑屏幕以及电脑内部声音。
	
#环境：
	windows 10 系统
	Python3.7.3

#相关包升级和安装
	python -m pip install --upgrade pip
	pip install -r requirement.txt

#使用：
	python record.py
	
#注意：
	1.录制的视屏和音频是分离的，由于码率不同，两者不同步。图像和音频存放在code所在目录。
	2.图像和音频可以使用adobe premiere 将两者合并，调节视屏的时间轴和音频的一至，导出媒体即可。
